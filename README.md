MachineLearning-Work repository overview
This repository contains a comprehensive collection of machine learning and data mining projects completed as part of Fundamental and Advanced Machine Learning coursework. The work demonstrates hands-on experience with data exploration, supervised learning, unsupervised learning, and advanced analytics, using R and Python, with a strong focus on business interpretation and reproducible analysis.

Repository Structure

The repository is organized to clearly distinguish between fundamental concepts and advanced machine learning applications:

├── FML/
│   └── Fundamental Machine Learning assignments
│      (EDA, basic modeling, model evaluation)
│
├── Assignment-1/
│   └── Exploratory Data Analysis & GitHub workflow
│
├── Assignment-2/
│   └── k-Nearest Neighbors (k-NN) Classification
│
├── Assignment-3/
│   └── Naive Bayes Classification
│
├── Assignment-4/
│   └── Customer Segmentation using K-Means Clustering
│
├── Final_Project/
│   └── Advanced Machine Learning / Applied Analytics Project
│
└── README.md
